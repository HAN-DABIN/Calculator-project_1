# Calculator-project_1
CH 2 계산기 과제

*****

![img.png](img.png)

### ✅ 계산기 성능
1. **사칙 연산**(➕,➖,✖️,➗)을 할 수 있습니다.
2. 계산된 **결과값**을 볼 수 있습니다.
3. 계산된 값 중 첫 번째 결과를 **삭제**할 수 있습니다.

### 🚨 유의 사항
1. **양의 정수**를 입력해주십시오.
   <br>➡️음의 정수 입력 시 <span style="color:red">재입력</span>이 필요합니다.
2. 숫자를 **0으로 나눌 수 없습니다.**
   <br>➡️ 나눗셈 다음 숫자를 0으로 입력 시 <span style="color:red">재입력</span>이 필요합니다.
